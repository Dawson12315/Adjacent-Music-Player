# Adjacent Music Player

Full-stack Spotify-style music app built with:

- FastAPI (backend)
- React (frontend)
- Local music library (Synology)

## Environment Variables:

- APP_NAME
- APP_ENV
- DEBUG
- DATABASE_URL
- MUSIC_LIBRARY_PATH
